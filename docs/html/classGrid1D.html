Detailed Description

Represents the grid as a 1D vector.

This class is a wrapper around a 1D vector that represents the grid. It provides methods to get and set the color of a cell.

Parameters
wThe width of the grid.
hThe height of the grid.
Note
We use an initialization list to initialize the members of the class, so we improve performance. Also, we initialize the data vector with -1, which represents an uncolored cell.
We don't use a class destructor because the vector will be destroyed automatically when the object is destroyed, following the Rule of Zero (Classes without manual resource management shouldn't have a user-defined destructor).

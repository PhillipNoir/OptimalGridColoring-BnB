◆ SelectNextNodeMRV()

int SelectNextNodeMRV ( const SearchState &  state)

Selects the next node to color using the Minimum Remaining Values (MRV) heuristic.

  • Scans uncolored nodes and returns the index of the node that has the fewest valid color options available. This ensures we tackle the most restrictive nodes first to fail fast and prune early.
  • Parameters
    stateThe current search state.
    Returns
    The 1D index of the selected node.

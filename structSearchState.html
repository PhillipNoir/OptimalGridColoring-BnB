Detailed Description

Encapsulates the entire current context of the search, including the grid state, color tallies, and the best solution found.

Parameters
wThe width of the grid.
hThe height of the grid.
Note
We use an initialization list to initialize the members of the class, so we improve performance.
We initialize the colorCounts vector with 5, we'll ignore the first index to avoid using substraction operations in every cycle.
We initialize the uncoloredNodes vector with w * h, which represents the total number of nodes.
We initialize the currentBestDiff with the maximum value of an int, so any valid diff will be smaller.
